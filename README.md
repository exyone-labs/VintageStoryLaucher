# VSL

Vintage Story 服务器启动器（Windows / WPF）。

## 功能概览

- 版本安装与本地 ZIP 导入
- 档案管理与配置编辑
- 存档管理与 Mod 管理
- 控制台日志、命令发送、日志下载

## 运行与构建

```powershell
dotnet build VSL.sln -c Release
```

## 打包

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\package.ps1 -Version 1.0.0
```
